# Test Audio Data

Large audio samples are stored on the deployment/test server and are not committed to Git.

Expected local filenames for existing test scripts:

- `teacher1.wav`
- `test.aac`
- `test1_16k.pcm`
- `test2_16k(1).pcm`
- `test3.pcm`

Place these files in this directory before running the audio stress or websocket tests.
